UTSOR stand for Unable To Synchronize On Requestor.
UTSOR Smasher is an utility which sole purpose is to find out why an UTSOR was generated.

UTSOR support PRPC 7.1.8, PRPC 7.1.9, PRPC 7.2, PRPC 7.2.1, PRPC 7.2.2.
Support for previous PRPC version will be added.
INSTRUCTION: go to File->Scan Folder and provide a folder which contains both Pega logs and Pega Alert logs. That's it!

How it works:
UTSOR Smasher will scan PegaRules log file for any UTSOR alert, then will scan PegaRules Alert logs for any related alert. Any alert related to the same affected user/thread will be returned.
Yellow lines corresponds to alert generated during the typical 2 minutes hang before the Alert generated.
Red lines corresponds to alert of high value and should contain pertinent information for this issue (based on extremely complicated algorithm :))
